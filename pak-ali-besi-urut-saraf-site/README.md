## Upload to GitHub

1. Extract the ZIP file.
2. Open the `luckyfeet93/pak-ali-besi-urut-saraf` repository.
3. Select **Add file → Upload files**.
4. Upload the **contents inside this folder**, not the ZIP itself.
5. Commit the files to the `main` branch.

## Deploy with Vercel

1. Open the Vercel dashboard.
2. Select **Add New → Project**.
3. Import `luckyfeet93/pak-ali-besi-urut-saraf`.
4. Set **Framework Preset** to `Other`.
5. Leave Build Command and Output Directory blank.
6. Select **Deploy**.

Vercel will redeploy automatically after each commit to the `main` branch.
